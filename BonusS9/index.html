Bonus Soal 1

Kode awal error karena ada dua extension pada String dengan nama method sama (toInt()), sehingga Dart bingung memilih method yang digunakan.

Solusinya: gunakan nama method berbeda, seperti toInt() dan toDouble().

//Nama: Ratu Klarisa Rahadianti 
//NIM: 251410010
//Kelas: SI2A

extension StringNumber on String {
  int toInt() {
    return int.parse(this);
  }
}

extension StringDouble on String {
  double toDouble() {
    return double.parse(this);
  }
}

void main() {
  String angka = '123';
  String desimal = '12.5';

  print(angka.toInt());      // 123
  print(desimal.toDouble()); // 12.5
}

Bonus Soal 2

Kode gagal dikompilasi karena extension CounterStats tidak bisa mengakses _value secara langsung. Field _value bersifat private, sehingga hanya bisa diakses di dalam class Counter.

Solusi:

Buat method public di dalam class sebagai perantara, lalu extension memanggil method tersebut.

//Nama: Ratu Klarisa Rahadianti 
//NIM: 251410010
//Kelas: SI2A
  
class Counter {
  int _value = 0; // private, tidak bisa diakses langsung oleh extension

  void increment() {
    _value++;
  }

  // method public sebagai perantara untuk mengambil nilai _value
  int getValueInternal() {
    return _value;
  }
}

extension CounterStats on Counter {
  int getValue() {
    // extension memanggil method public, bukan akses _value langsung
    return getValueInternal();
  }
}

void main() {
  var c = Counter();
  c.increment();
  print(c.getValue()); 
}
