Resume Materi Sesi 3

Pada sesi ini dibahas mengenai constructor dan encapsulation pada bahasa pemrograman Dart. Constructor adalah method khusus yang digunakan untuk melakukan inisialisasi objek saat objek dibuat.

Dart memiliki berbagai jenis constructor seperti default constructor, named constructor, parameterized constructor, dan constant constructor.

Encapsulation merupakan teknik untuk melindungi data dengan menggunakan field private yang ditandai dengan underscore (_).

Akses terhadap data dilakukan melalui getter dan setter agar data lebih aman dan terkontrol.

Slide 14 : Ringkasan

  • Dart memiliki berbagai jenis constructor.
  • Enkapsulasi melindungi data dengan private field.
  • Getter dan setter memberikan kontrol akses yang aman.

class RekeningBank {

 String _nomorRekening;
 double _saldo;

 RekeningBank(this._nomorRekening,this._saldo);

 void deposit(double jumlah){
  _saldo += jumlah;
 }

 void tarik(double jumlah){
  if(jumlah <= _saldo){
   _saldo -= jumlah;
  }else{
   print("Saldo tidak cukup");
  }
 }

 void cekSaldo(){
  print("Saldo saat ini : $_saldo");
 }

}
