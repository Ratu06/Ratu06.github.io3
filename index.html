Resume Materi

Pada sesi ini dipelajari mengenai constructor dan encapsulation pada bahasa pemrograman Dart. Constructor adalah method khusus yang digunakan untuk menginisialisasi objek saat objek dibuat.

Dart memiliki beberapa jenis constructor seperti default constructor, named constructor, parameterized constructor, dan constant constructor.

Encapsulation adalah teknik untuk menyembunyikan data di dalam class menggunakan variabel private yang ditandai dengan underscore (_).

Untuk mengakses data private digunakan getter dan setter sehingga data dapat diatur dan divalidasi.

Bonus Soal 1 - Class RekeningBank

class RekeningBank {

 String _nomorRekening;
 double _saldo;

 RekeningBank(this._nomorRekening,this._saldo);

 void deposit(double jumlah){
  if(jumlah > 0){
   _saldo += jumlah;
   print("Deposit berhasil");
  }
 }

 void tarik(double jumlah){
  if(jumlah <= _saldo){
   _saldo -= jumlah;
   print("Penarikan berhasil");
  }else{
   print("Saldo tidak cukup");
  }
 }

 void cekSaldo(){
  print("Saldo saat ini: $_saldo");
 }

}
