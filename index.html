🔁 Review Sesi 1

OOP berfokus pada objek. Kelas adalah blueprint, objek adalah instance. Constructor digunakan untuk inisialisasi.

class Buku {
  String judul;
  Buku(this.judul);
}

🏗️ Jenis Constructor

  • Default: ClassName()
  • Named: ClassName.name()
  • Parameterized: ClassName(this.param)
  • Constant: const ClassName()
  • Factory: (dibahas lanjut)

🚗 Named Constructor – Contoh Mobil

import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized constructor
  Mobil(this.merk, this.tahun);

  // Named constructor tanpa parameter
  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  // Named constructor dari JSON
  Mobil.dariJson(Map<String, dynamic> json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  @override
  String toString() => 'Mobil{merk: $merk, tahun: $tahun}';
}

⚡ Constant Constructor (Immutable)

Semua field harus final. Menghemat memori untuk objek yang sama.

class Warna {
  final int r, g, b;
  const Warna(this.r, this.g, this.b);

  @override
  String toString() => 'Warna(r: $r, g: $g, b: $b)';
}

void main() {
  const merah = Warna(255, 0, 0);
  const merahLagi = Warna(255, 0, 0); // instance yang sama
  print(merah);
}

📋 Constructor dengan Initializer List

Inisialisasi sebelum body, berguna untuk final field atau validasi.

class Titik {
  final double x, y;
  Titik(double x, double y)
      : x = x.abs(),
        y = y.abs() {
    print('Titik dibuat di ($x, $y)');
  }
}

🧑‍🎓 Praktik 1 – Multiple Constructor (Mahasiswa)

Buat kelas dengan tiga constructor berbeda.

class Mahasiswa {
  String? nama;
  int? angkatan;

  // default
  Mahasiswa();

  // dari Map
  Mahasiswa.dariMap(Map<String, dynamic> map) {
    nama = map['nama'];
    angkatan = map['angkatan'];
  }

  // tanpa nama (set default)
  Mahasiswa.tanpaNama() {
    nama = 'Anonim';
    angkatan = 2024;
  }
}

🔒 Enkapsulasi & Private Field

Menyembunyikan detail implementasi dengan underscore _. Private hanya dalam satu file.

class Person {
  String _name;   // private
  int age;        // public

  Person(this._name, this.age);
}

📥 Getter & Setter (Dua Bentuk)

class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  // Getter gaya 1
  String get getName1 => _name;

  // Getter gaya 2 (method)
  String getName() => _name;

  // Setter gaya 1
  set setName1(String name) {
    _name = name;
  }

  // Setter gaya 2
  void setName(String name) {
    _name = name;
  }
}

✅ Setter dengan Validasi (Produk)

class Produk {
  String _nama;
  double _harga;

  Produk(this._nama, this._harga);

  // Getter
  String get nama => _nama;
  double get harga => _harga;

  // Setter dengan validasi
  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Gagal: Nama minimal 3 karakter');
    }
  }

  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Gagal: Harga harus > 0');
    }
  }
}

👤 Praktik 2 – Kelas User (Enkapsulasi Lengkap)

Field private _username, _password, _email dengan validasi.

class User {
  String _username;
  String _password;
  String _email;

  User(this._username, this._password, this._email);

  // Getter
  String get username => _username;
  String get password => _password; // (hanya contoh, sebaiknya tidak diekspos)
  String get email => _email;

  // Setter dengan validasi
  set username(String value) {
    if (value.length >= 5) _username = value;
    else print('Username minimal 5 karakter');
  }

  set password(String value) {
    if (value.length >= 8) _password = value;
    else print('Password minimal 8 karakter');
  }

  set email(String value) {
    if (value.contains('@')) _email = value;
    else print('Email harus mengandung @');
  }
}

🔐 Studi Kasus: Sistem Login Sederhana

class User {
  String _username;
  String _password;

  User(this._username, this._password);

  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print('Password berhasil diubah.');
      } else {
        print('Gagal: Password baru minimal 8 karakter.');
      }
    } else {
      print('Gagal: Password lama salah.');
    }
  }
}

📌 Ringkasan & Latihan Mandiri

  • Dart memiliki berbagai jenis constructor (default, named, constant, dll).
  • Enkapsulasi menggunakan _ untuk private field.
  • Getter/setter memberikan kontrol akses dan validasi.

🏦 Latihan: Kelas RekeningBank

Buat kelas dengan:

  • Field private: _nomorRekening, _saldo
  • Method deposit(), tarik(), cekSaldo()
  • Validasi: saldo tidak boleh negatif
  • Simulasi transaksi sederhana
// Contoh kerangka jawaban
class RekeningBank {
  String _nomorRekening;
  double _saldo = 0;

  RekeningBank(this._nomorRekening);

  void deposit(double jumlah) {
    if (jumlah > 0) _saldo += jumlah;
  }

  bool tarik(double jumlah) {
    if (jumlah > 0 && _saldo >= jumlah) {
      _saldo -= jumlah;
      return true;
    }
    return false;
  }

  double get cekSaldo => _saldo;
}
