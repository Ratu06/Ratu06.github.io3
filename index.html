Review Sesi 1: OOP adalah paradigma berfokus pada objek. Kelas = blueprint, objek = instance. Constructor digunakan untuk inisialisasi objek.

Jenis Constructor di Dart:

  • Default Constructor: ClassName()
  • Named Constructor: ClassName.name()
  • Parameterized Constructor: ClassName(this.param)
  • Constant Constructor: const ClassName() (semua field final, immutable, hemat memori)
  • Factory Constructor: (lanjutan)

Enkapsulasi: Menyembunyikan detail implementasi dengan underscore _. Tujuan: keamanan, kontrol akses, fleksibilitas.

Getter & Setter: Memberikan akses terkontrol ke field private, bisa ditambahi validasi.

 class Buku {

  // Constructor
  Buku(this.judul);

  // Method untuk menampilkan data buku

    print("Judul buku: $judul");
  }
}

void main() {
  // Membuat objek dari class Buku
  Buku buku1 = Buku("Belajar Dart");

Named Constructor-Contoh Mobil

Named constructor adalah constructor yang memiliki nama tambahan selain nama class. Constructor ini digunakan untuk membuat lebih dari satu cara membuat objek dalam satu class sehingga proses inisialisasi bisa berbeda sesuai kebutuhan.

 //Nama: Ratu Klarisa Rahadianti
//Nim: 251410010
//Kelas: SI2A


import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized constructor
  Mobil(this.merk, this.tahun);

  // Named constructor
  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  // Named constructor dengan parameter
  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  // Override toString for better print output
  @override
  String toString() {
    return 'Mobil{merk: $merk, tahun: $tahun}';
  }
}

void main() {
  print('--- Demonstrasi Class Mobil ---');

  // 1. Menggunakan constructor default (parameterized constructor)
  var mobilA = Mobil('Honda', 2020);
  print('Mobil A (constructor default): $mobilA');

  // 2. Menggunakan named constructor Mobil.baru()
  var mobilB = Mobil.baru();
  print('Mobil B (named constructor .baru): $mobilB');

  // 3. Menggunakan named constructor Mobil.dariJson()
  String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
  Map jsonData = jsonDecode(jsonString);
  var mobilC = Mobil.dariJson(jsonData);
  print('Mobil C (named constructor .dariJson): $mobilC');

  // Contoh lain dengan named constructor Mobil.dariJson()
  Map anotherJsonData = {'merk': 'Mitsubishi', 'tahun': 2022};
  var mobilD = Mobil.dariJson(anotherJsonData);
  print('Mobil D (named constructor .dariJson): $mobilD');

  print('\n--- Akhir Demonstrasi ---');
}

constant contructor-Warna

Constant constructor adalah constructor yang digunakan untuk membuat objek yang bersifat konstan (tidak berubah). Constructor ini menggunakan kata kunci const sehingga nilai atribut pada objek harus tetap dan tidak dapat diubah setelah dibuat. Constant constructor biasanya digunakan untuk menghemat memori, karena jika ada objek dengan nilai yang sama maka Dart akan menggunakan objek yang sudah ada.

 //Nama: Ratu Klarisa Rahadianti
//Nim: 251410010
//Kelas: SI2A


class Warna {
  final int r, g, b;
  const Warna(this.r, this.g, this.b);

  @override
  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const merah = Warna(255, 0, 0);
  const merahLagi = Warna(255, 0, 0); // Menggunakan instance yang sama di memory

  print(merah);
  print(merahLagi);
}

Initializer List-Titik

Initializer list adalah cara untuk menginisialisasi nilai variabel sebelum constructor dijalankan. Pada Dart, initializer list ditulis menggunakan tanda titik dua (:) setelah constructor. Cara ini biasanya digunakan untuk mengisi variabel final atau melakukan perhitungan awal sebelum objek dibuat.

//Nama: Ratu Klarisa Rahadianti
//Nim: 251410010
//Kelas: SI2A


class Titik {
  final double x, y;

  Titik(double x, double y)
    : x = x.abs(),
      y = y.abs() {
    print('Titik dibuat di (${this.x}, ${this.y})'); // Using this.x and this.y to show the absolute values
    
  }
}

void main(){
  // Contoh penggunaan class Titik
  print('Membuat objek Titik pertama:');
  var titik1 = Titik(3.0, 4.0); // x=3.0, y=4.0
  print('Titik 1: x = ${titik1.x}, y = ${titik1.y}');

  print('\nMembuat objek Titik kedua dengan nilai negatif:');
  var titik2 = Titik(-5.0, 10.0); // x=5.0 (abs), y=10.0 (abs)
  print('Titik 2: x = ${titik2.x}, y = ${titik2.y}');

  print('\nMembuat objek Titik ketiga dengan nilai negatif di kedua koordinat:');
  var titik3 = Titik(-1.5, -2.5); // x=1.5 (abs), y=2.5 (abs)
  print('Titik 3: x = ${titik3.x}, y = ${titik3.y}');
}

Enkapsulasi

Konsep: Menyembunyikan detail implementasi

Tujuan: Keamanan, kontrol akses, fleksibilitas.

Di Dart gunakan _(undescore) untuk private member

Publik VS Private

Public dan Private adalah konsep dalam pemrograman berorientasi objek yang digunakan untuk mengatur tingkat akses terhadap variabel atau method dalam sebuah class. Public berarti atribut atau method dapat diakses dari luar class. Private berarti atribut atau method hanya dapat diakses di dalam class itu sendiri. Dalam bahasa Dart, variabel atau method dianggap private jika diawali dengan tanda underscore (_).

//Nim: 251410010
//Kelas: SI2A


class Person {
  String _name;
  int age;
  Person(this._name, this.age);
  
}

void main() {
  Person person = Person("Ratu", 17);
  print(person.age);
  // print(person._name); // error karena private
  
}

Contoh Setter & Getter

Getter dan Setter adalah method yang digunakan untuk mengambil (getter) dan mengubah (setter) nilai variabel private dalam sebuah class. Konsep ini digunakan untuk menjaga keamanan dan kontrol akses data, sehingga variabel tidak dapat diakses langsung dari luar class. Getter → digunakan untuk mengambil nilai dari variabel private. Setter → digunakan untuk mengubah atau memberikan nilai baru pada variabel private.

//Nama: Ratu Klarisa Rahadianti
//Nim: 251410010
//Kelas: SI2A


class Produk {
  String _nama;
  double _harga;
  
  Produk(this._nama, this._harga);
  
  // Getter
  String get nama => _nama;
  double get harga => _harga;
  
  // Setter dengan validasi
  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Gagal memperbarui nama: Nama harus memiliki minimal 3 karakter.');
    }
  }
  
  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Gagal memperbarui harga: Harga harus lebih besar dari 0.');
    }
  }
}

void main() {
  // Membuat instance dari class Produk
  var produk1 = Produk('Monitor Samsung', 3500.0);

  print('--- Informasi Produk Awal ---');
  print('Nama Produk: ${produk1.nama}');
  print('Harga Produk: Rp ${produk1.harga.toStringAsFixed(2)}');

  // Menggunakan setter untuk memperbarui nilai
  print('\n--- Memperbarui Produk ---');
  produk1.nama = 'Keyboard Mekanik';
  produk1.harga = 1200.0;

  print('Nama Produk (setelah update): ${produk1.nama}');
  print('Harga Produk (setelah update): Rp ${produk1.harga.toStringAsFixed(2)}');

  // Menguji validasi setter
  print('\n--- Menguji Validasi Setter ---');
  
  // Nama yang tidak valid (kurang dari 3 karakter)
  print('Mencoba mengatur nama menjadi "TV"');
  produk1.nama = 'TV'; // Ini seharusnya tidak mengubah nama
  print('Nama Produk (setelah gagal update nama): ${produk1.nama}');

  // Harga yang tidak valid (kurang dari atau sama dengan 0)
  print('Mencoba mengatur harga menjadi -100.0');
  produk1.harga = -100.0; // Ini seharusnya tidak mengubah harga
  print('Harga Produk (setelah gagal update harga): Rp ${produk1.harga.toStringAsFixed(2)}');

  print('Mencoba mengatur harga menjadi 0.0');
  produk1.harga = 0.0; // Ini seharusnya tidak mengubah harga
  print('Harga Produk (setelah gagal update harga): Rp ${produk1.harga.toStringAsFixed(2)}');

  // Memastikan nilai tidak berubah setelah percobaan yang tidak valid
  print('\n--- Status Akhir Produk ---');
  print('Nama Produk: ${produk1.nama}');
  print('Harga Produk: Rp ${produk1.harga.toStringAsFixed(2)}');
}

Buat kelas User dengan: i.Field private: _username _password, _email ii.Getter untuk semua field iii.Setter dengan validasi: Username minimal 5 karakter Password minimal 8 karakter Email harus mengandung '@' Buat objek dan coba modifikasi data.

//Nama: Ratu Klarisa Rahadianti

class User {
  String _username;
  String _password;
  String _email;

  // Constructor
  User(this._username, this._password, this._email);

  // Getter
  String get username => _username;
  String get password => _password;
  String get email => _email;

  // Setter dengan validasi
  set username(String value) {
    if (value.length >= 5) {
      _username = value;
    } else {
      print("Username minimal 5 karakter.");
    }
  }

  set password(String value) {
    if (value.length >= 8) {
      _password = value;
    } else {
      print("Password minimal 8 karakter.");
    }
  }

  set email(String value) {
    if (value.contains('@')) {
      _email = value;
    } else {
      print("Email harus mengandung '@'.");
    }
  }

  void info() {
    print("Username: $_username");
    print("Password: $_password");
    print("Email   : $_email");
  }
}

void main() {
  var user1 = User("Riska", "password2525", "Riska@gmail");

  user1.info();

  print("---- Ubah Data ----");

  user1.email = "Riska80ea@email.com";

  user1.info();
}

class User {
  String _username;
  String _password;

  // Constructor
  User(this._username, this._password);

  // Getter username
  String get username => _username;

  // Method login
  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  // Method ganti password
  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print("Password berhasil diubah.");
      } else {
        print("Gagal mengubah password: Password baru minimal 8 karakter.");
      }
    } else {
      print("Gagal mengubah password: Password lama salah.");
    }
  }
}

void main() {
  // Membuat user
  User user1 = User("ratuUser", "12345678");

  print("=== SISTEM LOGIN SEDERHANA ===");

  // Simulasi login
  if (user1.login("ratuUser", "12345678")) {
    print("Login berhasil");
  } else {
    print("Login gagal");
  }

  // Mengganti password
  user1.gantiPassword("12345678", "passwordBaru");

  // Login lagi dengan password baru
  if (user1.login("ratuUser", "passwordBaru")) {
    print("Login dengan password baru berhasil");
  } else {
    print("Login gagal");
  }
}

Latihan Kelas Rekening Bank (Bonus 3)

Buat kelas RekeningBank dengan:

Field private: _nomor Rekening, saldo

Method: deposit(), tarik(), cekSaldo()

Validasi: saldo tidak bisa negatif

Buat program sederhana untuk simulasi transaksi.

//Nama: Ratu Klarisa Rahadianti
//Nim: 251410010
//Kelas: SI2A

class RekeningBank {
  String _nomorRekening;
  double _saldo;

  // Constructor
  RekeningBank(this._nomorRekening, this._saldo);

  // Method deposit
  void deposit(double jumlah) {
    if (jumlah > 0) {
      _saldo += jumlah;
      print("Deposit Rp $jumlah berhasil.");
    } else {
      print("Jumlah deposit tidak valid.");
    }
  }

  // Method tarik
  void tarik(double jumlah) {
    if (jumlah > 0 && jumlah <= _saldo) {
      _saldo -= jumlah;
      print("Penarikan Rp $jumlah berhasil.");
    } else {
      print("Penarikan gagal. Saldo tidak cukup atau jumlah tidak valid.");
    }
  }

  // Method cek saldo
  void cekSaldo() {
    print("Nomor Rekening : $_nomorRekening");
    print("Saldo Saat Ini : Rp $_saldo");
  }
}

void main() {
  // Membuat objek rekening
  RekeningBank rekening = RekeningBank("123456789", 1000000);

  print("=== SIMULASI TRANSAKSI BANK ===");

  rekening.cekSaldo();
  print("----------------------");

  // Simulasi deposit
  rekening.deposit(500000);
  rekening.cekSaldo();
  print("----------------------");

  // Simulasi penarikan
  rekening.tarik(300000);
  rekening.cekSaldo();
  print("----------------------");

  // Simulasi penarikan gagal
  rekening.tarik(2000000);
  rekening.cekSaldo();
}
