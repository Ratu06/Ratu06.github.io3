Soal 1. ConnectionFactory dengan Pooling Koneksi Database

ConnectionFactory merupakan konstruktor pabrik (factory constructor)

yang berfungsi untuk membuat sekaligus mengelola koneksi ke database.

Apabila koneksi ke host dan port yang sama telah tersedia sebelumnya

sistem akan menggunakan kembali objek koneksi yang sudah ada daripada membuat koneksi baru.

Cara ini membantu mengurangi penggunaan memori dan membuat kinerja aplikasi menjadi lebih efisien.

//Nama: Ratu Klarisa Rahadianti 
//NIM: 251410010
//Kelas: SI2A

class ConnectionFactory {
  final String host;
  final int port;

  static final Map<String, ConnectionFactory> _cache = {};

  ConnectionFactory._(this.host, this.port);

  factory ConnectionFactory(String host, int port) {
    String key = '$host:$port';

    if (_cache.containsKey(key)) {
      print('Menggunakan koneksi yang sudah ada');
      return _cache[key]!;
    }

    print('Membuat koneksi baru');
    var connection = ConnectionFactory._(host, port);
    _cache[key] = connection;

    return connection;
  }

  void connect() {
    print('Terhubung ke $host:$port');
  }
}

void main() {
  var c1 = ConnectionFactory('localhost', 3306);
  var c2 = ConnectionFactory('localhost', 3306);

  c1.connect();

  print(identical(c1, c2));
}

Soal 2. NotificationFactory Memilih Subclass Berdasarkan Platform

NotificationFactory adalah factory constructor yang berperan

dalam membuat objek notifikasi sesuai dengan platform yang

digunakan, seperti Android, iOS, maupun Web. Dengan

mekanisme ini, proses pembuatan objek menjadi lebih mudah

karena sistem akan secara otomatis menentukan dan menghasilkan

subclass yang sesuai tanpa perlu dibuat secara manual oleh pengguna.

//Nama: Ratu Klarisa Rahadianti 
//NIM: 251410010
//Kelas: SI2A

abstract class NotificationFactory {
  void send(String message);

  factory NotificationFactory(String platform) {
    switch (platform.toLowerCase()) {
      case 'android':
        return AndroidNotification();

      case 'ios':
        return IOSNotification();

      case 'web':
        return WebNotification();

      default:
        throw ArgumentError('Platform tidak dikenal');
    }
  }
}

class AndroidNotification implements NotificationFactory {
  @override
  void send(String message) {
    print('Android Notification: $message');
  }
}

class IOSNotification implements NotificationFactory {
  @override
  void send(String message) {
    print('iOS Notification: $message');
  }
}

class WebNotification implements NotificationFactory {
  @override
  void send(String message) {
    print('Web Notification: $message');
  }
}

void main() {
  var notif1 = NotificationFactory('android');
  notif1.send('Halo Android');

  var notif2 = NotificationFactory('ios');
  notif2.send('Halo iOS');
}

Soal 3. ShapeFactory dengan Cache Shape yang Sama

ShapeFactory merupakan factory constructor yang digunakan untuk menghasilkan objek bentuk (shape),

seperti lingkaran dan persegi. Factory ini memanfaatkan mekanisme cache sehingga jika objek dengan

jenis yang sama sudah pernah dibuat, objek tersebut dapat digunakan kembali tanpa perlu membuat objek baru.

Pendekatan ini membantu menghemat memori dan meningkatkan efisiensi aplikasi.

//Nama: Ratu Klarisa Rahadianti 
//NIM: 251410010
//Kelas: SI2A

class ShapeFactory {
  final String shape;

  static final Map<String, ShapeFactory> _cache = {};

  ShapeFactory._(this.shape);

  factory ShapeFactory(String shape) {
    if (_cache.containsKey(shape)) {
      print('Menggunakan shape dari cache');
      return _cache[shape]!;
    }

    print('Membuat shape baru');

    var newShape = ShapeFactory._(shape);
    _cache[shape] = newShape;

    return newShape;
  }

  void draw() {
    print('Menggambar $shape');
  }
}

void main() {
  var s1 = ShapeFactory('Lingkaran');
  var s2 = ShapeFactory('Lingkaran');

  s1.draw();

  print(identical(s1, s2));
}

Soal 4. AnimalFactory (Challenge)

AnimalFactory merupakan factory constructor yang berfungsi untuk membuat objek

hewan sesuai dengan masukan yang diberikan pengguna. Factory ini dapat melakukan pengecekan

validitas data, menentukan jenis subclass yang tepat secara otomatis, serta menyimpan objek yang telah

dibuat ke dalam cache. Dengan demikian, objek yang sudah tersedia dapat digunakan kembali tanpa perlu dibuat ulang,

sehingga proses menjadi lebih efisien.

//Nama: Ratu Klarisa Rahadianti 
//NIM: 251410010
//Kelas: SI2A

abstract class AnimalFactory {
  String name;

  AnimalFactory(this.name);

  void sound();

  static final Map<String, AnimalFactory> _cache = {};

  factory AnimalFactory.create(String type, String name) {
    if (name.isEmpty) {
      throw ArgumentError('Nama hewan tidak boleh kosong');
    }

    String key = '$type-$name';

    if (_cache.containsKey(key)) {
      print('Mengambil hewan dari cache');
      return _cache[key]!;
    }

    AnimalFactory animal;

    switch (type.toLowerCase()) {
      case 'cat':
        animal = Cat(name);
        break;

      case 'dog':
        animal = Dog(name);
        break;

      case 'bird':
        animal = Bird(name);
        break;

      default:
        throw ArgumentError('Jenis hewan tidak dikenal');
    }

    _cache[key] = animal;

    return animal;
  }
}

class Cat extends AnimalFactory {
  Cat(String name) : super(name);

  @override
  void sound() {
    print('$name berkata: Meow');
  }
}

class Dog extends AnimalFactory {
  Dog(String name) : super(name);

  @override
  void sound() {
    print('$name berkata: Guk Guk');
  }
}

class Bird extends AnimalFactory {
  Bird(String name) : super(name);

  @override
  void sound() {
    print('$name berkata: Cuit Cuit');
  }
}

void main() {
  var a1 = AnimalFactory.create('cat', 'Milo');
  var a2 = AnimalFactory.create('cat', 'Milo');

  a1.sound();

  print(identical(a1, a2));
}

Kesimpulan Poin Penting

Ringkasan

- ConnectionFactory berfungsi untuk membuat dan mengatur koneksi database sehingga koneksi yang sudah ada dapat digunakan kembali secara efisien.

- NotificationFactory digunakan untuk menentukan dan menghasilkan objek notifikasi yang sesuai dengan platform yang digunakan.

- ShapeFactory berperan dalam mengelola objek shape dengan memanfaatkan cache agar objek dengan tipe yang sama tidak perlu dibuat berulang kali.

- AnimalFactory digunakan untuk menghasilkan objek hewan berdasarkan input pengguna, dilengkapi dengan validasi data, pemilihan subclass secara otomatis, serta penyimpanan objek dalam cache untuk meningkatkan efisiensi.
